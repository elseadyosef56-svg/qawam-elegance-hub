# المصدر الاصلي — Al-Masdar Al-Asli Food Website

موقع تعريفي ثابت (Static) لشركة **المصدر الاصلي للمواد الغذائية** — مبني بـ HTML5 و CSS3 و Vanilla JavaScript فقط، يدعم العربية (RTL) ومصمم بأسلوب Apple العصري والبسيط، متجاوب بالكامل (Mobile-first).

## ✨ المميزات
- تصميم Premium بأسلوب Apple
- دعم كامل للغة العربية واتجاه RTL
- خطوط Google (Cairo + Tajawal)
- متجاوب 100% (Mobile-first)
- قائمة جانبية للهاتف (Hamburger Menu)
- شريط تنقل ثابت (Sticky Navbar)
- Smooth Scrolling + Animations
- Lazy Loading للصور
- نموذج تواصل (Frontend فقط)

## 🚀 الاستخدام
ببساطة افتح ملف `index.html` في المتصفح مباشرة.

## 🌐 النشر على GitHub Pages
1. أنشئ Repository جديد على GitHub باسم `almasdar-food-website`.
2. ارفع جميع ملفات المشروع إلى الفرع `main`.
3. اذهب إلى **Settings → Pages**.
4. تحت **Source** اختر `Deploy from a branch`.
5. اختر الفرع `main` والمجلد `/ (root)` ثم اضغط **Save**.
6. خلال دقائق سيظهر رابط الموقع:
   `https://<your-username>.github.io/almasdar-food-website/`
